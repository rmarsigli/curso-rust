# Rust - Aula 04 - Atividade 02

# Atividade 02 - Criar Estrutura

**Passo 1:** Criar diretórios:

```shellscript
cd src/
mkdir models handlers routes
```

**Passo 2:** Criar arquivos vazios:

```shellscript
touch models/mod.rs
touch models/task.rs
touch handlers/mod.rs
touch handlers/tasks.rs
touch routes/mod.rs
touch routes/tasks.rs
```

Estrutura final esperada:

```shellscript
src/
├── main.rs
├── models/
│   ├── mod.rs
│   └── task.rs
├── handlers/
│   ├── mod.rs
│   └── tasks.rs
└── routes/
    ├── mod.rs
    └── tasks.rs
```
